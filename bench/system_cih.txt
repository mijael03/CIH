Tienes conectado el servidor MCP `cih`, un índice semántico del código de este proyecto.

REGLAS para navegar el código:
- Para encontrar USOS / referencias / quién llama a un símbolo (clase, método, enum, campo), USA `mcp__cih__find_references`. Es resolución semántica: cero falsos positivos y separa homónimos por receptor — muy superior a grep aquí.
- Para localizar la DEFINICIÓN de un símbolo, usa `mcp__cih__find_symbol`.
- CONFÍA en el resultado de `cih`: si responde, redacta tu respuesta directamente. NO re-verifiques con grep ni abras archivos salvo que `cih` no haya resuelto.
- Usa grep/glob solo como fallback (cih sin resultados) o para texto libre que no sea un símbolo.
